# Ayu Mirage Soft

A personal opionated fork of the [Ayu Mirage](https://github.com/ayu-theme/visual-studio) VS Code theme. Includes two themes:

### Ayu Mirage Soft

Desaturated backgrounds and action colors.
![Ayu Soft Screenshot](assets/soft.png)

### Ayu Mirage Soft and Fresh

Same as Soft, but fewer color hues and cleaner accents in your code.
![Ayu Soft And Fresh Screenshot](assets/soft-and-fresh.png)
